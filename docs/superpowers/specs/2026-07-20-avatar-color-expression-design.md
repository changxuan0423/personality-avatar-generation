# 头像批次配色与表情强度设计

## 目标

修复「美式漫画半调风格头像」的两个稳定性问题：同一批次背景颜色重复，以及人物表情被压到缺少美式社论漫画感。其他已确认的青年年龄、人物抽象、自然工作构图、低饱和旧报刊印刷语言和参考图机制保持不变。

## 批次背景配色

- 采用批次预排色盘。生成第一张前，先为本批次全部角色分配背景色。
- 同一批次每张背景必须视觉上不同；不能只改颜色名称，却保留近似色相、冷暖和明度。
- 优先跨色相族分配：灰紫、鼠尾草绿、暖米黄、灰蓝、柔和砖粉、浅橄榄、灰青、低饱和杏色。角色数量更多时，再通过冷暖和明度扩展，但仍要肉眼可区分。
- 背景色先确定，随后为服装、设备和强调色选择协调关系。低饱和与明度层级继续生效，禁止随机霓虹色和等强度撞色。
- `06-positive-ai-knows-you.png` 只控制低饱和协调关系，不控制具体背景色。单张生成不复制其灰紫背景；连续系列沿用当前批次已用色清单。

## 表情强度

- 将表情设为五档中的第 3 档：漫画性清晰可见，但不进入戏剧化讥讽。
- 默认组合两个协调线索：一个主要线索与一个辅助线索。例如眉形＋嘴角、眼神收束＋轻微头部倾斜、眼睛张力＋单侧面颊变化。
- 表情随角色变化，可使用好奇、机敏、自信、专注、轻度怀疑或轻微得意；不能让所有角色复用同一半笑。
- `07-negative-overexpressive-ai-knows-you.png` 只定义上限：排除强挑眉、强眯眼、撅嘴、明显坏笑和戏剧化侧目，不再把输出推回几乎中性。

## 验收

1. Skill 明确要求生成前建立批次背景色分配表。
2. Skill 明确要求同批次背景视觉上全部不同。
3. Prompt 明确说明 `06` 不控制具体背景色。
4. 表情规则明确为第 3 档，并允许两个协调线索。
5. 自检能识别重复背景和过弱/过强表情。
6. README 的能力摘要不再把表情描述为过度克制。

## 不在范围内

- 不改变五图输入上限。
- 不生成新的示例头像。
- 不清理 `/neat` 报告中的备份、分支或工作区候选。
