# 人格头像生成

本仓库收录两个并列的 Codex 人物头像生成 skills。

| Skill | 适用方向 |
| --- | --- |
| `everme-digital-avatar-generation/` | EverMe 品牌数字头像：真实人物基础、复古社论漫画与专属参考资产。 |
| `screenprint-character-illustration/` | 丝网印刷人物插画：紧凑桌前构图、密集半调网点与可变角色系列。 |

每个目录均可独立使用。将所需 skill 目录复制到 `~/.codex/skills/` 后，重新开启 Codex 会话即可调用。

## 调用示例

```text
使用 EverMe 数字头像生成 skill，生成角色：营养顾问
```

```text
使用 screenprint-character-illustration skill，生成一个新的桌前单人物丝网印刷插画
```

旧的 `everme-digital-avatar-generation.tar.gz` 保留为原 EverMe skill 的兼容安装包。
